# Daniel Meireles Portfolio

Personal portfolio built to present my projects, technical background and contact links.

The website has a clean interface, light and dark themes, Portuguese and English content, smooth animations and project previews.

## Tech Stack

* Next.js
* TypeScript
* Tailwind CSS
* Vercel

## Running Locally

```bash
npm install
npm run dev
```

Open `http://localhost:3000` in your browser.

## Project Structure

```txt
src/
  app/          pages and global styles
  components/   UI sections and reusable components
  context/      theme and language providers
  data/         portfolio content

public/         images, resume and static files
```

## Content

The main portfolio content is managed in `src/data/content.ts`.
